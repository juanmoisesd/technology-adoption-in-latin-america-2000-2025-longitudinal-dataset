# Consent & Data Use Statement

## Dataset: Longitudinal dataset (2000-2025) on digital technology adoption indicators across 21 Latin American and Iberian countries. Variables: internet penetration, mobile subscriptions, smartphone adoption, social media usage, e-commerce, digital payments, A

**Author:** Juan Moisés de la Serna Tuya | ORCID: https://orcid.org/0000-0002-8401-8018

---

## 1. Nature of Data

This dataset contains **aggregated, anonymized data** compiled exclusively from **publicly available secondary sources**. No primary data collection involving human subjects was conducted for this compilation.

## 2. Ethical Basis

- No individual consent was required (aggregate/public data only)
- All original data sources are publicly available and properly attributed
- This compilation follows open data principles (CC0 1.0)

## 3. Permitted Uses

Users of this dataset may:
- Use the data freely for research, education, and policy analysis
- Redistribute and modify the data under CC0 1.0 terms
- Cite the dataset in academic and non-academic publications

## 4. Restrictions

- Users should not attempt to re-identify individuals from aggregate data
- Commercial use is permitted but attribution is encouraged
- Misuse for discrimination or harm is explicitly discouraged
