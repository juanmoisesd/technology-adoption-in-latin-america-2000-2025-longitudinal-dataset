# Frequently Asked Questions (FAQ)

## About: Longitudinal dataset (2000-2025) on digital technology adoption indicators across 21 Latin American and Iberian countries. Variables: internet penetration, mobile subscriptions, smartphone adoption, social media usage, e-commerce, digital payments, A

---

**Q: Is this data free to use?**  
A: Yes. Released under CC0 1.0 Universal — no restrictions.

**Q: How do I cite this dataset?**  
A: See CITATION.cff or CITATION.bib in the root folder.

**Q: How accurate is the data?**  
A: Compiled from authoritative secondary sources. See docs/validation.md.

**Q: Can I contribute or report errors?**  
A: Yes! Open an issue on GitHub or contact juanmoises.delaserna@unir.net

**Q: Is there a DOI?**  
A: Check the Zenodo record linked in the README.

A: CSV, HTML. See data/ folder.
