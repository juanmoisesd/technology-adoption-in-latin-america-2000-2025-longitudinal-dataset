### Key Terms

**CC0 (Creative Commons Zero)**
A public domain dedication that waives all copyright and related rights, allowing free use without restrictions.

**CITATION.cff**
Citation File Format — a standardized YAML file that allows automatic citation generation in GitHub and Zotero.

**codemeta.json**
A JSON-LD file with structured scientific metadata following the CodeMeta schema for software and datasets.

**DataCite**
An international organization that provides persistent identifiers (DOIs) for research data.

**DOI (Digital Object Identifier)**
A persistent identifier for digital objects, commonly used for academic publications and datasets.

**FAIR Principles**
Findable, Accessible, Interoperable, Reusable — guidelines for scientific data management and stewardship.

**Open Data**
Data published online with an open license allowing free use, reuse, and redistribution.

**ORCID**
Open Researcher and Contributor ID — a persistent digital identifier for researchers.

**RO-Crate**
Research Object Crate — a method of packaging research data with their metadata using Schema.org.

**Zenodo**
An open-access repository operated by CERN for research outputs including datasets, software, and publications.
