# Ethics Statement

## Research Ethics

This dataset/project adheres to the following ethical principles:

### Data Sources
- All data is compiled from **publicly available secondary sources**
- No primary data collection involving human subjects was conducted for this specific compilation
- Original data sources have been properly attributed

### Privacy
- No personally identifiable information (PII) is included
- Data is aggregated at population/country level
- All individual-level data (if any) has been anonymized

### Transparency
- Full methodology is documented in `docs/methodology.md`
- Data limitations are documented in `docs/limitations.md`
- Provenance is tracked in `PROV.json`

### Responsible Use
- This dataset is released under CC0 1.0 to maximize scientific reuse
- Users are encouraged to cite original sources
- The author disclaims responsibility for misuse of the data
