# Privacy Policy

## Data Privacy Statement

This repository contains **aggregated, anonymized data** compiled from public sources.

### No Personal Data

- No personally identifiable information (PII) is stored
- No individual-level data is included
- All data is at aggregate/population level

### Data Sources

All data is sourced from publicly available datasets and publications. Original sources are documented in the README and methodology files.
